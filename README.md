# Weather App

A simple, modern weather application that shows the current weather and a 5‑day forecast for any city using the OpenWeather API.

## Features

- Search for the weather by city name
- Current temperature, humidity, wind speed, and weather description
- 5‑day forecast with daily icons and min/max temperatures
- Responsive, card‑based UI built with HTML, CSS, Bootstrap, and JavaScript
- Axios for API requests

## Technologies Used

- HTML5
- CSS3 (custom styles)
- Bootstrap 5
- Vanilla JavaScript
- Axios
- OpenWeather API (Current Weather + 5‑day/3‑hour Forecast)

## How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser  
   **or** use a “Live Server” extension in your editor.
3. Type a city name and press **Search** to view its weather.

## API Setup

This project uses an OpenWeather API key that is hard‑coded in `src/app.js` for learning purposes.  
For production, you should store your API key in environment variables or a separate config file.
